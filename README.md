# Employee Attrition Analysis (Tableau)

## Project Overview
This project analyzes employee attrition using HR analytics data.
The goal is to understand key factors contributing to employee turnover.

## Tools Used
- Tableau Public
- Excel
- IBM HR Analytics Dataset

## Key KPIs
- Total Employees: 1470
- Employees Left: 237
- Attrition Rate: 16%

## Analysis Included
- Attrition by Department
- Attrition by Age Group
- Attrition by OverTime

## Dashboard
The Tableau dashboard provides HR-friendly insights to support data-driven retention strategies.
